# To-Do App 🚀

A simple and modern To-Do application built using HTML, CSS, and JavaScript.  
This app allows users to manage daily tasks efficiently with a clean and responsive UI.

---

## ✨ Features

- ✅ Add new tasks
- ✅ Mark tasks as completed
- ✅ Delete tasks
- ✅ Data saved using localStorage
- ✅ Tasks remain after page refresh
- ✅ Responsive modern UI

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Browser LocalStorage
